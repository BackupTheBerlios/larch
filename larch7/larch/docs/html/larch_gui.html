larch – a do-it-yourself live Arch Linux system



Using the GUI

larch-7 is (in contrast to previous versions) designed to be run from a graphical user interface. It is possible to use it from the command line, but this should really only be done when scripting builds - the command line provides just a subset of the operations available in the GUI, in a much less comfortable form.

The larch documentation is available in the GUI, and there are documentation pages for each of the tabs in the larch window. For convenience these are also linked from here.
